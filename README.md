# BridgeHours Starter Site

This is a lightweight starter website for a volunteer-hour platform that connects students with local people who need help.

## Run locally

```powershell
python server.py
```

Then open:

`http://127.0.0.1:8000`

## Current sections

- Mission-driven landing page
- Product flow overview
- Mock request-posting interface
- Filterable opportunity board
- Trust and safety section
- Next-step roadmap
